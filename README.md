# LP11 資料ダウンロードページ

士業特化型・生成AI活用術の資料ダウンロードランディングページです。

## 概要

社労士、税理士、行政書士など、士業の現場ですぐに使える業務別AI活用術と成功事例を提供するLPサイトです。

## ファイル構成

- `index.html` - メインページ（資料一覧、導入事例、運営会社情報）
- `download_*.html` - 各資料の詳細ページ（9ページ）
- `*.webp`, `*.jpg` - 画像ファイル
- `*.png` - ロゴ・アイコン

## 9つの資料ページ

1. 法改正案内メール自動生成プロンプト
2. SEO対策ブログ記事作成プロンプト
3. 補助金自動リストアップ方法
4. 顧客対応履歴要約プロンプト
5. 相続税簡易試算プロンプト
6. マニュアル質問対応bot作成方法
7. よくある質問回答テンプレート生成プロンプト
8. セミナースライド自動作成方法
9. ビジョンを計画書に落とし込む方法
